Download / Java Web Start

Java Web Start (a Sun trademark) is a convenient way to start Java applications from the Net. The application archives are kept in a local cache and will be used in off-line mode. Java Web Start automatically looks for updates of the application, so updating is no longer a problem.

You will need at least Java 1.2 to use Java Web Start. Supported operating systems are all flavors of Windows, Linux and Sun's Sparc. Simulum needs at least Java 1.4.

Follow the following steps.

  • Java Web Start is included in Java 1.4. If you have no Java version or you are using an earlier version, download and install Java Web Start from Sun's page, if necessary.
  • Then click on mulumis 0.00.03. There should be a splash screen for Java Web Start, asking you to trust the signature. Although it will be not recommended you must trust the signature of Simulum to start the application.
  • If you want to use the program off-line, you may now use a link to Java Web Start directly. Java Web Start can also install an icon on the desktop for you.

For a preview for the upcoming new release just click on 0.04.44.
